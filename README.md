# AI Blockchain Protocol

This project is a blockchain testnet built with AI integration, wallet system, mining, economy, and explorer.

## Features
- Core blockchain (blocks, transactions, chain validation)
- Mining with difficulty and rewards
- Economy: fees, burn, staking, supply, dynamic price
- Wallet with RSA keys and signing
- Persistence with JSON
- Explorer API (Express)
- Web UI with AI helper

## Getting Started
```bash
npm install
npm start
